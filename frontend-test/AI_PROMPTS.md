# AI Prompts Documentation

This document includes the main AI prompts used during the development of this project.

---

## 1. Project Setup
"Create a React + TypeScript project using Vite with a clean folder structure and routing setup."

---

## 2. Authentication (Login Page)
"Build a login page in React using DummyJSON API (POST /auth/login). Handle authentication, store token, and redirect to dashboard on success."

---

## 3. Dashboard & API Integration
"Fetch products from DummyJSON API (GET /products) and display them in a responsive dashboard table using React and TypeScript."

---

## 4. Filtering
"Add a search input to filter products by title in real time using React state and JavaScript filter method."

---

## 5. Sorting
"Implement sorting functionality for product price column with ascending and descending toggle using React state."

---

## 6. Draggable Columns
"Implement draggable table columns in React using HTML drag and drop API and dynamically reorder columns using state."

---

## 7. Product Modal
"Create a modal popup in React that displays full product details when clicking on a table row."

---

## 8. Responsive Design
"Make the dashboard responsive for mobile, tablet, and desktop using CSS and flexible layout techniques."

---

## 9. Debugging
"Fix React routing issues on Netlify by adding _redirects file and resolving build configuration problems in Vite."

---

## Tools Used
- React
- TypeScript
- Vite
- DummyJSON API
- Netlify
- Git & GitHub
- AI-assisted development tools
